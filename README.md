# 知乎日报

# 简介

知乎日报的第三方iOS客户端，没有第三方库依赖，使用Objective-C语言实现。
第一个iOS项目，轻拍...
API来源：[知乎日报 API 分析](https://github.com/izzyleung/ZhihuDailyPurify/wiki/%E7%9F%A5%E4%B9%8E%E6%97%A5%E6%8A%A5-API-%E5%88%86%E6%9E%90)

# 所用技术

1. MVC
2. GCD
3. 自定义tableview cell
4. 图片轮播插件
5. 缓存
6. UI Animation
7. Objective-C Runtime特性
